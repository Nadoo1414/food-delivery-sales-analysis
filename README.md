# 🍽️ Food Delivery Sales Analysis (Swiggy Dataset)

## 📌 Project Overview

This project analyzes food delivery (Swiggy) sales data using Python to uncover key business insights such as revenue trends, top-performing restaurants, and customer ordering patterns.

## 🎯 Objectives

* Analyze overall sales performance
* Identify top restaurants and popular items
* Understand customer behavior
* Visualize trends for better decision-making

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn / Plotly
* Jupyter Notebook

## 📊 Key Analysis Performed

* Total revenue and sales trends
* Top-performing restaurants
* Most ordered food items
* Monthly / daily sales patterns
* Data visualization using graphs and charts

## 📁 Project Structure

* `data/` → Contains dataset
* `notebooks/` → Jupyter Notebook for analysis
* `README.md` → Project documentation

## 🚀 How to Run

1. Install required libraries:

   ```
   pip install pandas numpy matplotlib seaborn plotly
   ```
2. Open Jupyter Notebook:

   ```
   jupyter notebook
   ```
3. Run the notebook file:

   ```
   swiggy_sales_analysis.ipynb
   ```
